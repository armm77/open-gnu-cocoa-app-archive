ToyViewer (Ver.4.10)

ToyViewer can make and attach a custom icon to a file.
Note that the size of the file increases because a custom icon needs 40 - 60 KBytes.

ToyViewer's custom icon functions are not implemented currently for DOS formatted volumes.

Attach Icon to Newly Saved File

When you save an image into new file, check the button "Make Custom Icon (Thumbnail)" on the save panel in order to attach an icon.

If no area is selected on the image, the whole image is shrinked to make an icon. If there is a selected area, it is used to make an icon.

Attach Icon to Selected Files

ToyViewer can make a custom icon from a displayed image and then can attach it to existing files.
First, display an image, and select area if needed. Then, Click "File / Attach Custom Icon..." of the menu to show an open panel. Select files to which you want to attach a custom icon. You can select more than one files, or directories.

Remove Custom Icon

ToyViewer can remove attached custom icons from files.
Click "File / Remove Custom Icon..." of the menu to show an open panel. Select files from which you want to remove custom icons. You can select more than one files.
