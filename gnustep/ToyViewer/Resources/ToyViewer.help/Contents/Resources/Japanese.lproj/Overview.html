ToyViewer (Ver.4.30)

機能概要

  1. 以下の画像フォーマットを表示できます。
    tiff, gif, bmp, png, jpg, bie(jbig), pcx, pcd, pict, pnm(ppm, pbm, pgm), xbm, mag, SUN Rasterfile, JPEG2000(jp2, jpc, j2k),
    および Mac OS X　がサポートしているその他の画像フォーマット。
  2. 画像を tiff, pdf, gif, bmp, png, jpg, JPEG2000, bie(jbig), pnm, あるいは xbm形式で保存できます。
  3. 画像の保存と同時に、そのファイルにカスタムアイコンを添付できます。また、任意のファイルにカスタムアイコンを添付できます。
  4. 透明色を持つgif, png、およびインタレース(プログレッシヴ)形式のgif, png, jpgが読み書きできます。
  5. フォルダを指定して、その中のファイルを順番に自動表示できます。また、フルスクリンーでの表示が可能です。
  6. 表示した画像を拡大・縮小できます。
  7. 画像を回転、反転させたり、一部分を選択して切り出したりできます。
  8. 輝度やコントラスト、色調の調節のほか、エッジ強調、エンボスなど、いくつかの代表的な画像加工ができます。
  9. 指定した色を他の色、あるいは透明色で置換できます。
  10. 256, 64, 8色への減色、4, 2, 1ビットへのビット切り捨てができます。
  11. モノクロ画像(8ビット・2ビットグレイ、白黒２値)に変換できます。
  12. 画像の印刷ができます。
  13. どんな画像ファイルでも ppm形式へのフィルタがあれば、ToyViewerで表示するように設定できます。
  14. 他のアプリケーションに対して、画像変換のフィルタサービスを提供します。
  15. 画像にコメントを付加できます(ただし、コメントが保存できるのはgif, png, jpg, pnm形式のみ)。
  16. Aquaのボタン風の画像を作ることができます。

注意事項

ToyViewer 4.30 のバイナリ配布版は Mac OS X 10.2（およびそれ以降）上でのみ動作します。10.1.x 上で動作させるには、ソースコードからコンパイルして下さい。

現バージョン 4.30 にはまだ次のような不具合があります（Mac OS X 10.2 で動作の場合）。

  • EPSが正しく表示できない (Mac OS X が未対応)
  • GIFで正しく保存できない (Mac OS X が未対応)。フィルタプログラムが必要です。
  • 読み込めないTIFF画像がある (JPEG圧縮を使ったTIFFはMac OS Xでサポート外となった)
  • 画像変換サービスを正しく受けられないアプリケーションがある (調査中)
