ToyViewer (Ver.4.30)

画像ファイルの表示

以下の形式の画像ファイルは、Finder の「ファイル/情報を見る」(Cmd-I) で設定することで、ファイルビューア上のアイコンをダブルクリックしてオープン可能です。
tiff, (eps), gif, bmp, png, jpg, JPEG2000(jp2, jpc, j2k), bie(jbig), pcx, pcd, pnm, ppm, pbm, pgm, xbm, mag, Sun Raster
EPSファイルの表示は、現在の Mac OS X (10.2) では実現されていません。

ToyViewer の「ファイル/オープン」メニューからファイルを指定してオープンすることもできます。

また、Dock、Finder、あるいはデスクトップに表示された ToyViewer のアイコン上に画像ファイルのアイコンをドラッグ＆ドロップすることによってもオープンできます。
画像ファイルによっては、ファイルの拡張子が ToyViewerで用意したものと異なる場合もあります。特に、Mac OS で作成されたファイルには拡張子が付けられておらず、クリエータが指定されている場合が多くあります。 そのような場合には、ドラッグ＆ドロップでファイルをオープンすることができます。

画像の保存

表示させた画像を別形式のファイルとして保存できます。保存可能な形式は以下の通りです。
tiff, (eps), pdf, bmp, gif, png, jpg, JPEG2000, bie(jbig), pnm, xbm
gif形式は最大256色の画像しか保存できません。それ以上の色数を持つ画像を保存しようとすると、自動的に減色操作が行われます。 bie(jbig)形式、xbm形式として保存できるのは、モノクロ２値画像だけです。

ある形式を他の形式に変換する際、通常は表示時の縮尺は作成される画像サイズに影響しません。ただし、eps形式, pdf形式のファイルを他の形式で保存する場合は表示時の縮尺によって画像のサイズが変わります。

png および bmp 形式には、パレットを使って画像をコンパクトに保存する形式と、各ピクセルごとに色を指定する形式があります。ToyViewerでこれらの形式に保存する際、自動的に色数が調べられ、可能であればパレット形式として保存されます。

ペーストボードからオープン

tiff形式、pdf形式、あるいは eps形式の画像が（コピー操作によって）ペーストボードにコピーされている場合、メニューの「ファイル/ペーストボードからオープン」をクリックすればその画像を新しいウィンドウ内に表示できます。 例えば、OmniGraffleで描いた図形の一部をコピーしてからこの操作を行うと、その図形を pdf画像として表示できます。また、QuickTime Playerでムービーの一場面をコピーすれば、それをpict画像として表示できます。

拡大・縮小表示

ウィンドウの右下にあるポップアップリストで倍率を指定することによって、画像の大きさを変化させることができます。ポップアップリストに倍率（パーセント）を直接書き込むことによって、任意の大きさを指定できます。
拡大・縮小の機能は画面上の表示の大きさを変えるもので、画像データ自体を変更するものではありません。大きさを変化させた画像を新たに作りたい場合には、「画像の操作」で説明している「拡大・縮小」を使って下さい。

リマークボタン

ウィンドウの右下には小さなチェックボタンがあります。これはリマークボタンと言います。
数多くの画像を表示した後で、それらを「全部のウィンドウをクローズ」という操作でクローズできますが、このボタンがチェックされているウィンドウだけは自動的にクローズされません。
また、フォルダ内の自動表示という機能を使うと、画像を次々に表示させ、いったん見た画像は自動的にクローズして行くことができます。この時も、リマークボタンがチェックされていればウィンドウが自動的にクローズされることはありません。

画像の印刷

表示させた画像を印刷できます。

ページ設定パネルのポップアップメニュー「設定」に "ToyViewer" という項目が用意されています。これを利用して、プリンタの解像度、用紙のマージンを設定することができます。
解像度を高く設定するほど、印刷される画像のサイズは小さくなります。 多くのプリンタはドットの密度によって色の濃さを調節していますので、解像度はカタログ値より数倍低く設定した方が良いでしょう。
